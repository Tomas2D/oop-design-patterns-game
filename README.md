# How to start?

## With docker

```
docker build -t oop-design-patterns-game .
docker run -p 8000:80 -it oop-design-patterns-game # app will be available on http://localhost:8000
```

For delete, simply run: `docker rmi oop-design-patterns-game`

## How to run test?

```
yarn test
```
